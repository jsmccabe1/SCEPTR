The Method

Three statistical ideas that reveal what standard approaches miss.

▲

Multi-Resolution Enrichment Profiling

Genes ranked by TPM are evaluated at nested tiers (top 50, 100, 250, 500). At each tier, functional category proportions are compared to the transcriptome-wide background using Fisher's exact test with Benjamini-Hochberg correction. The result is an enrichment profile: a curve showing how each programme's dominance changes across the expression gradient.

DKL

KL Divergence as Functional Specialisation

Kullback-Leibler divergence at each tier measures how different the functional composition is from the background. This decreases monotonically with tier size across every organism tested. The rate of decrease is itself a quantitative phenotype: steep gradients indicate extreme functional concentration, shallow gradients indicate distributed investment.

✦

Dual-Method Category Assignment

Genes are assigned to functional categories through keyword regex matching against UniProt annotations and GO hierarchy traversal from curated anchor terms. Each assignment is tagged with its source. GO-only ablation recovers 100% of enriched categories (mean r = 0.90), confirming keywords are supplementary.

◉

Bootstrap Stability

Bootstrap resampling (default 50 iterations) assesses whether enrichment estimates are robust to perturbation of gene composition. Each result carries a stability score and Clopper-Pearson confidence intervals so you know which findings to trust.

⇄

Cross-Sample Comparison

Compare enrichment profiles between two conditions (e.g. mock vs infected) using gene-label permutation testing. Tests whether observed profile differences exceed what would be expected by chance, with per-category significance and effect size.

☉

Single Sample. No Replicates.

Because each tier is compared to the sample's own background, SCEPTR works from a single sample with no replicates, no control, and no comparative data. The clinical isolate, the field sample, the pilot experiment. If you have expression data, SCEPTR tells you what your transcriptome is investing in.
