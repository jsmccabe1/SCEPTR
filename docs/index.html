The Method

A continuous enrichment framework that reveals what standard approaches miss.

▲

Continuous Enrichment Profiling

Genes ranked by TPM are evaluated at every tier from k=10 to the full transcriptome in configurable steps, computing a continuous enrichment function EC(k) for each functional category. Fisher's exact test with Benjamini-Hochberg correction at each tier produces smooth enrichment profiles that reveal how each programme's dominance changes across the entire expression gradient - not just at a few arbitrary thresholds.

DKL

KL Divergence as Functional Specialisation

Kullback-Leibler divergence computed across the continuous enrichment profile measures how functional composition diverges from the background at every resolution. This decreases monotonically with tier size across every organism tested. The rate of decrease is itself a quantitative phenotype: steep gradients indicate extreme functional concentration, shallow gradients indicate distributed investment.

✦

Dual-Method Category Assignment

Genes are assigned to functional categories through keyword regex matching against UniProt annotations and GO hierarchy traversal from curated anchor terms. Each assignment is tagged with its source. GO-only ablation recovers 100% of enriched categories (mean r = 0.90), confirming keywords are supplementary.

◉

Permutation Profile Testing

A global profile test permutes gene-category labels (default 1000 iterations) to assess whether the observed continuous enrichment profile for each category is significantly non-random. Each result carries a permutation p-value so you know which findings reflect genuine transcriptomic investment rather than annotation noise.

⇄

Cross-Sample Comparison

Compare enrichment profiles between two conditions (e.g. mock vs infected) using gene-label permutation testing. Tests whether observed profile differences exceed what would be expected by chance, with per-category significance and effect size.

☉

Single Sample. No Replicates.

Because each tier is compared to the sample's own background, SCEPTR works from a single sample with no replicates, no control, and no comparative data. The clinical isolate, the field sample, the pilot experiment. If you have expression data, SCEPTR tells you what your transcriptome is investing in.

What You Get

From raw reads to biological insight in a single self-contained report.

Interactive Report
A single HTML file containing enrichment profiles, significance testing, and full results tables. Self-contained and shareable - open it in any browser, no additional software required.
Transcriptome Overview
Interactive Lorenz curve, expression distribution, annotation quality by expression decile, and taxonomic composition - a complete summary of your transcriptome's structure before any enrichment analysis.
Continuous Enrichment Profiles
Visualise how each functional programme rises and falls across the full expression gradient. Identify where investment peaks, where it drops off, and which programmes dominate at which expression levels.
Built for Any Transcriptome
Whether you work with clinical isolates, environmental metagenomes, agricultural pathogens, cancer cell lines, or coral symbionts - SCEPTR characterises functional investment from whatever expression data you have.
Automated End-to-End

SCEPTR's methodology is implemented in a fully automated pipeline. From raw FASTQ reads to an interactive HTML report in a single command, with Nextflow and Docker for full reproducibility.
